# Примеры промптов для запуска разработки через агента

Промпт итерировался три раза — каждая версия исправляла конкретный недостаток предыдущей.

---

## Версия 1 — первый черновик

```
Cформируй верхнеуровневый план реализации MVP по проекту
```

**Проблема:** агент не знает, какой проект, с какого старта и в каком формате нужен результат. Вернул общий план без привязки к контексту.

---

## Версия 2 — добавлен стартовый контекст

```
Cформируй верхнеуровневый план реализации MVP по проекту.
Пока у нас даже не инициализирован проект Rails.
```

**Проблема:** агент учёл точку старта, но не читал PROJECT.md и придумал функциональность сам. Также не зафиксировал результат — план остался только в чате.

---

## Версия 3 — финальная

```
Cформируй верхнеуровневый план реализации MVP по проекту.

Context:
- описание проекта см. в файле PROJECT.md
- на текущий момент у нас даже не инициализирован проект Rails

Constraints:
- для каждого шага плана должны быть указаны критерии оценки успешного выполнения шага

Output:
- создай .claude/memory/ и сохрани план там в формате .md
```

**Результат:** агент прочитал PROJECT.md, составил план из 5 фаз с 22 задачами, для каждой задачи прописал критерий приёмки, сохранил в `.claude/memory/plans/general.md`.
