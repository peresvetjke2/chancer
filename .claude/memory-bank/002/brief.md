# Issue #2: Load Team Matches

## Стейкхолдер

AI-помощник по прогнозированию — система, которая отвечает на свободные запросы пользователя о форме команд и формирует прогнозы на предстоящие матчи.

## Проблема

В системе отсутствуют данные о матчах команд за последний месяц. Без истории матчей AI-помощник не может оценить текущую форму команды и выдать обоснованный прогноз.

## Желаемый результат

По любой команде в системе доступны матчи за последний месяц.

## Контекст

Задача блокирует дальнейшее развитие проекта: без истории матчей невозможно реализовать ни аналитику формы команд, ни модуль прогнозирования. Это фундаментальный слой данных для всей системы.
