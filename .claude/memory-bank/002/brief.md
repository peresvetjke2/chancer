# Issue #2: Load Team Matches

## Проблема

В системе отсутствуют актуальные данные о недавних матчах команд, необходимые для расчёта формы команды. Без этих данных невозможно корректно анализировать и прогнозировать выступления в предстоящих матчах.

## Требования

### 1. Загрузка матчей
- Запрос матчей по идентификатору команды (ID, slug или название)
- Фильтрация по временному периоду (последние N дней или последние N матчей)

### 2. Хранение данных
Каждый матч должен содержать:
- Дата, турнир, формат матча
- Команды (Team A / Team B)
- Результат (счёт, победитель)
- Информация по картам

### 3. Идемпотентность
- Исключить дубликаты матчей в базе данных
- Повторная загрузка тех же данных не должна приводить к дублированию

## Контекст

Проект связан со спортивной (киберспортивной) аналитикой. Загрузка истории матчей — основа для расчёта формы команды при анализе предстоящих игр.
