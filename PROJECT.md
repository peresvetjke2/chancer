AI-помощник для прогнозирования ставок на кибер-спорт.

## Цели сервиса

- Осуществлять сбор данных об истории игр команд и их состоянии, в т.ч. рейтингах, статистике, новостях и пр.
- Проводить анализ полученных данных
- Предоставлять ответы на свободные запросы пользователя
- Формировать прогнозы и обучаться на основе фактических результатов

## Функциональность

### Сбор данных
- Парсинг истории матчей команд (результаты, счёт, карты/раунды)
- Сбор актуальных составов и ростеров
- Интеграция с рейтинговыми системами (HLTV, Liquipedia и др.)
- Агрегация новостей и трансферов
- Мониторинг статистики игроков (K/D, рейтинг, винрейт)

### Анализ
- Аналитика формы команды (последние N матчей)
- Сравнение head-to-head двух команд
- Учёт карточного пула и win rate по картам
- Анализ домашней/выездной формы (LAN vs online)
- Оценка силы турнира/оппонентов

### Прогнозирование
- Генерация прогноза на конкретный матч с обоснованием
- Оценка вероятности победы в процентах
- Уровень уверенности / индикатор надёжности прогноза

### Интерфейс / запросы пользователя
- Свободный чат-интерфейс для запросов на естественном языке
- Поиск команды / игрока / турнира
- Лента предстоящих матчей с быстрым прогнозом

### Обратная связь и обучение
- Фиксация фактических результатов матчей
- Отслеживание точности прогнозов (accuracy, ROI)
- Корректировка весов модели на основе ошибок
- История прогнозов пользователя
